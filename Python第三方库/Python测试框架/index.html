Hypothesis

Hypothesis是基于属性的测试工具。简单地说，它会根据你提供的规范生成随机测试场景，直到找到导致测试失败的情况。在开始使用此工具之前，请花点时间了解背后的原理。

  • 用途：测试代码，尤其是数据处理
  • 场景：需要使用各种输入数（数字，字符串，结构化数据）测试复杂逻辑时
